# v0.01.0027 — 2026-07-09

执行者: Codex

## 新增

- 为 26 键新增前缀拼音匹配：当当前输入还不是完整拼音 key 时，会尝试从以该前缀开头的拼音 key 聚合候选词。
- 为 9 键新增数字前缀匹配：当当前数字串还不是完整映射时，会尝试从以该数字前缀开头的拼音 key 展示左侧拼音列表并聚合候选词。
- 前缀候选与前缀拼音 key 都加了结果上限，避免列表过大拖慢输入过程。

## 修改

- 完整拼音 / 完整数字串的原有行为保持不变：只在没有精确命中的情况下启用前缀匹配。
- 26 键前缀候选提交时，学习记录不再写入原始前缀文本，而是写入解析出的拼音 key。
- 更新显示版本号为 `v0.01.0027`。
- 更新 `PROJECT_HANDOFF.md`，将当前工作节点切换到 `v0.01.0027` 的前缀拼音匹配。
